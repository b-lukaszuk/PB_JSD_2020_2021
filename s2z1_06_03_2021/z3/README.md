# Tasks

## Task 3

Write a function that rotates a list by k elements. For example [1,2,3,4,5,6] rotated by two becomes [3,4,5,6,1,2]. Try solving this without creating a copy of the list.

## Uruchomienie programu

```bash
tsc main.ts  # transpilacja do JS-a
node main.js # uruchomienie progamu
```
opcjonalnie (do ES6)

```bash
tsc main.ts --target es6 # transpilacja do JS-a w wersji EcmaScript6
node main.ts # uruchomienie progamu
```
