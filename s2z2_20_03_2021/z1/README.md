# Przypomninie

## Uruchomienie programu

```bash
tsc main.ts  # transpilacja do JS-a
node main.js # uruchomienie progamu
```
opcjonalnie (do ES6)

```bash
tsc main.ts --target es6 # transpilacja do JS-a w wersji EcmaScript6
node main.ts # uruchomienie progamu
```
